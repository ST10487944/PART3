# Chrisoteffel Restaurant

A simple React Native app for managing a restaurant menu. Keep track of your dishes, prices and courses all in one.

# What does it do
1. You can see all the menu items on the home screen
2. You can add new dishes to your menu
3. It shows all starters, mains and desserts
4. Can view average prices for each course
5. You can remove dishes which you don't want anymore

# Screens
1. Home Screen: Show everything with cool statics
2. Add Screen: Add new food items to your menu
3. Filter Screen: Filter to see only certain courses
